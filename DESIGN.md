# 포트폴리오 디자인 문서 (Minimalist React Portfolio)

## 1. 개요
이 프로젝트는 React(TypeScript)와 순수 CSS(Vanilla CSS)를 사용하여 구축된 개인 포트폴리오 웹사이트입니다. 화이트와 그레이 톤을 활용한 깔끔하고 전문적인 "미니멀리스트" 스타일을 지향합니다.

## 2. 기술 스택
- **프레임워크:** React (Vite로 구성)
- **언어:** TypeScript
- **스타일링:** Vanilla CSS (모바일 우선 반응형 디자인)
- **라우팅:** 단일 페이지 내 스크롤 이동 (또는 필요시 React Router 도입)

## 3. 핵심 섹션 구성
1. **Header (내비게이션):** 상단 고정, 각 섹션으로 부드럽게 이동하는 링크 제공.
2. **Hero / About Me (자기소개):**
   - 크고 명확한 타이포그래피를 활용한 핵심 인사말.
   - 간단한 경력 및 목표 요약.
3. **Projects (프로젝트 갤러리):**
   - 그리드(Grid) 레이아웃을 사용한 프로젝트 목록.
   - 각 프로젝트의 스크린샷(또는 플레이스홀더), 제목, 설명, 사용 기술, GitHub/Live 링크 포함.
4. **Contact (연락처):**
   - 이메일 주소, LinkedIn, GitHub 링크 제공.
   - 깔끔한 텍스트 기반 레이아웃.
5. **Footer:** 심플한 저작권 표기 및 하단 여백.

## 4. 디자인 원칙 (Minimalist)
- **색상 팔레트:** 배경은 흰색(#FFFFFF) 및 아주 연한 회색(#F9F9F9), 텍스트는 짙은 회색(#333333) 및 검은색(#111111) 사용.
- **타이포그래피:** 가독성이 높은 Sans-serif 폰트 (예: Inter, Roboto 또는 시스템 폰트) 사용. 여백(Whitespace)을 충분히 활용하여 시각적 피로도 감소.
- **인터랙션:** 버튼 호버 시 부드러운 색상 전환(Transition), 스크롤 시 페이드 인(Fade-in) 등 과하지 않은 애니메이션 적용.
